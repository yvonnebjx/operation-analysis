# Report Outline Template

Use this template to draft `output/final/report_outline.md` before writing the full report.

This is a cross-border ecommerce operating-analysis outline, not a generic annual-report template.

## 1. 汇报目的与范围

Must state:

- analysis period: MTD and YTD
- budget-vs-actual and YoY scope
- data limitations that materially affect interpretation

## 2. 核心经营结论

Must answer:

- current month off-budget gap
- YTD off-budget gap
- scale versus unit-economics judgment
- top 2-4 structural risks

This section should fit on one screen or one slide.

## 3. 公司经营概况

Recommended subsections:

- KPI dashboard cards
- MTD KPI table
- YTD KPI table
- one short conclusion on revenue, gross profit, and gross margin trend

## 4. 利润差异拆解

Recommended subsections:

- GMV 价差量差
- 毛利瀑布图: MTD
- 毛利瀑布图: YTD
- key negative drivers by management category

## 5. 结构性风险

Recommended subsections:

- channel / region / category structure shift
- channel-region weak combinations
- category-channel weak combinations

## 6. 费用与退款风险

Recommended subsections:

- advertising / logistics / platform-cost pressure points
- refund risk by channel
- category and region refund follow-up inside each key channel

## 7. 异常产品或 MODEL

Recommended subsections:

- negative gross-profit models
- dominant driver categories
- immediate treatment suggestions: adjust, limit spend, clear inventory, or stop

## 8. 风险清单与待业务解释事项

Split into:

- proven business risks
- questions that specific departments must explain

## 9. 管理动作建议

Recommended split:

- immediate actions this month
- medium-term actions for next review cycle

## Appendix

Only include when needed:

- field mapping notes
- metric definitions
- reconciliation or data limitations
