# Analysis Methodology

This file defines how the agent should think about cross-border ecommerce operating analysis.

## Primary objective

Find valuable and abnormal business situations from data, then classify the risk in management terms.

## Analysis sequence

### 1. Company-level diagnosis

Answer first:

- how far actual is from budget
- how current month compares with last year
- whether the core issue is scale or unit economics

### 2. Profit decomposition

Always decompose profit variance into:

- volume effect
- selling price / mix effect
- discount effect
- refund effect
- purchase effect
- headship effect
- overseas warehouse effect
- advertising effect
- commission effect
- delivery fee effect
- storage fee effect
- other platform cost effect

### 3. Structural diagnosis

Check whether revenue growth is coming from profit-accretive or profit-dilutive structures.

Required dimensions:

- channel
- region
- category
- channel x region
- category x channel

### 4. Refund diagnosis

Refund should be analyzed by:

- company total
- channel
- category within channel
- region within channel

The point is not just to find the highest refund rate. The point is to find where refund deterioration is materially contributing to profit risk.

### 5. Abnormal product diagnosis

For abnormal models, classify issues as:

- volume shortfall led
- unit economics led
- mixed

Then identify the top negative drivers by profit impact.

## Risk classification

The agent should map abnormalities into business categories such as:

- scale shortfall
- selling price / mix deterioration
- discount deterioration
- refund deterioration
- purchase deterioration
- headship deterioration
- overseas warehouse deterioration
- advertising deterioration
- commission deterioration
- delivery fee deterioration
- storage fee deterioration
- other platform cost deterioration
- structural shift

## What not to do

Do not confuse:

- description of results
- management-level attribution
- operational root-cause explanation

This skill is responsible for the first two. The third often requires department follow-up.
